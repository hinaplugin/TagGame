# TagGame - 1.0.0

## Commands
```text
/taggame join
# 自分を鬼ごっこに参加させる

/taggame join <player>
# playerで指定したプレイヤーを鬼ごっこに参加させる

/taggame joinall
# オンラインのプレイヤー全員を鬼ごっこに参加させる

/taggame leave
# 鬼ごっこから退室する

/taggame start
# 鬼ごっこを開始する

/taggame set escape
# 逃走者のスタート位置を設定

/taggame set hunter
# 鬼のスタート位置を設定

/taggame set end
# 終了位置を設定

/taggame set size <size>
# 鬼の人数をsize人に設定

/taggame set time <time>
# 制限時間をtime分に設定

/taggame list
# 鬼と逃走者のリストを取得

/taggame stop
# 鬼ごっこを終了する
```
